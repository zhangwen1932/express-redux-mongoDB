# express-server-demo

##启动前端
```shell
npm run start
```

##启动后台服务端
```shell
npm run start-server
```

项目目录
```
.
├── README.md                   
├── app                     // 前端源码文件夹
│    └──index.js            // 入口文件 
├── package.json
├── record                  // 存放一些记录文件的文件夹
├── schemas                 // mongoose schema文件夹
├── server                  // server端源码文件夹
│   ├── server.js           // server文件
│   └── index.js            // server启动文件
├── static                  // 静态资源托管文件夹
└── webpack.dev.js          // webpack配置文件
```
